# Flappy Bird Game

A simple Flappy Bird clone built using HTML, CSS, and JavaScript.

## Features
- Gravity-based movement
- Pipe generation
- Collision detection
- Score counter
- Restart system
- Fullscreen mode

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
